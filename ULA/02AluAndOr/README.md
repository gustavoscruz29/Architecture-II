## 🧪 Simulation Details

- The input values are defined as:
  - **A = 1001**
  - **B = 0011**
- During the simulation, the operation control signal is changed to **1**, which corresponds to the **OR operation**.
- As a result, the output reflects the bitwise OR between **A** and **B**.
